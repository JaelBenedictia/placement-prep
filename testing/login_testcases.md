# Login Page Test Cases

| Test Case ID | Test Scenario | Steps | Expected Result |
|-------------|--------------|-------|----------------|
| TC01 | Valid Login | Enter valid username and password, click Login | User logs in successfully |
| TC02 | Invalid Password | Enter valid username and wrong password | Error message displayed |
| TC03 | Blank Fields | Leave username and password empty, click Login | Validation message shown |
| TC04 | Username Only | Enter username only and click Login | Password required message |
| TC05 | Forgot Password | Click Forgot Password link | Password reset page opens |
